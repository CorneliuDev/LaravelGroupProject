# Proiect Laravel - Gestiune Task-uri (CRUD)

Aplicație web dezvoltată conform specificațiilor pentru gestionarea activităților în echipă.

## Specificații Tehnice
- **Framework:** Laravel 11
- **Baza de date:** SQLite
- **Frontend:** Bootstrap 5
- **Gestiune Versiuni:** Git (SourceTree / GitHub)

## Membrii Echipei
1. **Membru 1 (Backend):** Implementare Model, Migrări și Controller.
2. **Membru 2 (Frontend):** Stilizare Bootstrap și Design Responsive.
3. **Membru 3 (Testing):** Testare interfață și validări.
4. **Membru 4 (Git Config):** Gestiune ramuri și integrare GitHub.

## Instrucțiuni Rulare Locală
1. Rulați `composer install`.
2. Copiați `.env.example` în `.env`.
3. Rulați `php artisan migrate`.
4. Porniți serverul cu `php artisan serve`.
